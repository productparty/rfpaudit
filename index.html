The Framework

Six phases. Zero guesswork.

Every phase must be completed in sequence. Skipping a level isn't efficiency — it's missed risk.

Phase 0

Go / No-Go

LPTA vs. Tradeoff identification, set-aside eligibility, NAICS verification, and procurement method analysis. Stop wasting effort on structurally ineligible bids.

Phase 1

RFP Intelligence

Agency hot button extraction, keyword frequency analysis, incumbent weakness signals, recompete context, and mandatory pre-submission requirements.

Phase 2

Section L Compliance

Binary pass/fail gate. Shall/must/will hierarchy sweep, formatting compliance, mandatory forms, security requirements, placeholder detection.

Phase 3

Threshold Verification

Past performance minimums, key personnel credentials, bonding capacity, CPARS verification, and 13 CFR 125.2 subcontractor rights analysis.

Phase 4

Adjectival Scoring

DoD Source Selection Procedures simulation. Factor-by-factor strength, weakness, and deficiency identification with documented "because" statements.

Phase 5

Discriminator Analysis

Generic language inventory, three-criterion discriminator test, competitive positioning, and the 18 evaluator questions from PropLibrary methodology.
